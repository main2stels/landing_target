# landing_target
<p>Запустить:</p>
<p>git clone https://github.com/main2stels/landing_target.git</p>
<p>cd landing_target/</p>
<p>sudo bash ./rosinstall.sh</p>
<h1>Параметр лист: </h1>
<p>PLND_ENABLED=1</p>
<p>RNGFND1_TYPE=10</p>
<p>SYSID_MYGCS=1</p>
<p>Перезагрузить полетный контроллер</p>
<p>PLND_ALT_MIN=0,3</p>
<p>PLND_EST_TYPE=0</p>
<p>PLND_LAG=0,1</p>
<p>PLND_TYPE=1</p>
<p>PLND_XY_DIST_MAX=10</p>
